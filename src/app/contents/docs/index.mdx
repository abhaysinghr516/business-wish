## Explore the library

Business Wish is a copy-paste component library for teams building with Tailwind CSS and React. Start with the core component collection, jump into motion primitives, or use complete page templates when you need a larger layout.

## Sections

- [Components](/docs/components): UI components and blocks for common product interfaces.
- [Motion](/docs/motion): Animated primitives powered by Framer Motion.
- [Pages](/docs/pages): Full-page templates and layout examples.
